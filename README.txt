How to set up the website:

1. 
Install WAMPServer. Go to http://www.wampserver.com/en/ and download WAMP installer.
2. Open the folder where you installed your WAMP.
3. There you will find a folder named www and open it.
4. Inside the folder, Create a new folder and name it Webtech.
5. Save all the files inside the Webtech folder.
6. Open any browser and type the link localhost/Webtech.
